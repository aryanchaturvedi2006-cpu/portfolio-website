About Me.

I'm a B.Tech student in Computer Science & Engineering with AI at JK Lakshmipat University. I'm driven by the challenge of building software that actually solves problems — not just demos.

My latest project CafeBuddy is a full-stack cafeteria application built with Next.js 15 and Tailwind CSS 4, featuring secure Google OAuth, real-time cart management with Zustand, and a seamless Razorpay payment integration.

My flagship project EduSync is a complete college management platform — multi-role login with Google OAuth, attendance tracking with threshold alerts, and GPA auto-calculation, built in Python and Flask.

Other works include IPL Predictor, which uses a live Node.js news scraper (Axios + Cheerio) and match prediction engines to provide real-time cricket analytics.

I believe the best way to learn is to build things that have real moving parts — OAuth flows, backend scrapers, database transactions, and UIs that feel polished. Every project is a chance to go deeper.
